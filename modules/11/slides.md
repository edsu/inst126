``` {.python}
>>> import pandas
>>>
>>> data = {
        'apples': [3, 2, 0, 1],
        'oranges': [0, 3, 7, 2]
    }
>>>
>>> df = pandas.DataFrame(data)
>>> df.head()
```

::: fragment
```
   apples  oranges
0       3        0
1       2        3
2       0        7
3       1        2
```
:::
